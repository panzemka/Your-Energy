# Your Energy

The project is built with vanilla JavaScript (ES modules), modern CSS, and Vite for fast development and optimized production builds. The layout is fully responsive and adapts smoothly to mobile, tablet, and desktop devices.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev      # Start dev server at localhost:5173
```
3. Build for production:

```bash
npm run build    # Production build to dist/
```
4. Preview production build:

```bash
npm run preview  # Preview production build
```
